# Agroculture
AgroCulture: Your Product, Our Market – A web application that connects farmers and buyers, enabling farmers to list products and buyers to browse, add to cart, and place orders.

AgroCulture is a web-based platform designed to connect farmers directly with buyers, enabling farmers to list their agricultural products and buyers to browse, select, and purchase them through an easy-to-use online marketplace.

The system helps eliminate middlemen, improve transparency in pricing, and create a direct digital marketplace for agricultural products.

# Features
# Farmer Dashboard
* Farmers can log in to their accounts
* Add new agricultural products
* Manage product listings
* View uploaded products

# Buyer Dashboard
* Buyers can log in and browse available products
* View product details
* Add products to cart
* Place orders easily

# Authentication System
* Separate login system for farmers and buyers
* Secure user authentication
* Role-based access control

# Product Management
* Farmers can upload:
* Product name
* Price
* Quantity
* Product details

# Cart System
* Buyers can add products to a shopping cart
* Selected products are displayed in the cart
* Buyers can place orders

# System Workflow
* Farmers register and log into the platform.
* Farmers add agricultural products with price and quantity.
* Buyers log into the system.
* Buyers browse available products.
* Buyers add products to the cart.
* Buyers place orders for selected products.
